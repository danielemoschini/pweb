# pweb
Progetto per l'esame di Progettazione Web. \n
L'idea è di creare un gioco basandosi sui Cellular Automata di Wolfram monodimensionali ->
https://www.wolframscience.com/nks/ . 
Lo scopo del gioco (wip) è di ricreare l'automata mostrato a schermo gestendo le regole di evoluzione, per 10 (variabile) round.
