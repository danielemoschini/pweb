# pweb
Progetto per l'esame di Progettazione Web.
L'idea è di creare un gioco basandosi sui Cellular Automata di Wolfram monodimensionali ->
https://www.wolframscience.com/nks/ . 
Lo scopo del gioco è di ricreare l'automata mostrato a schermo gestendo le regole di evoluzione, per un numero di round variabile in base alla difficoltà selezionata (5, 10, 15).
Il sito ha anche una pagina leaderboard per mostrare i punteggi migliori degli utenti e la cronologia delle partite, e una pagina per visualizzare tutte le possibili comnbinazioni di regole e automi.
